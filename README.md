# Rahul Kumar - Interactive Developer Portfolio

A modern, highly interactive, and visually stunning web developer portfolio designed to showcase skills, projects, and professional experience. Built with cutting-edge web technologies, it features dynamic WebGL shaders, fluid animations, and a rich, responsive user interface.

## 🚀 Live Demo
[https://master.d13kkfmbue8ihx.amplifyapp.com/](https://master.d13kkfmbue8ihx.amplifyapp.com/)

## ✨ Key Features
- **Dynamic WebGL Backgrounds**: Uses `Three.js` and custom fragment shaders to create a thick, bright, interactive rainbow background effect.
- **Fluid UI Animations**: Leveraging `tw-animate-css` and Tailwind classes to provide micro-interactions and smooth page transitions.
- **Premium Design System**: Built on top of `Shadcn UI` and `Radix UI` for accessible, beautiful, and customizable components.
- **Fully Responsive**: Optimized for seamless viewing across mobile, tablet, and desktop devices.
- **Modern Stack**: Developed with Next.js 16 (App Router), React 19, and TypeScript for robust performance and type safety.

## 🛠️ Technology Stack
- **Framework**: [Next.js](https://nextjs.org/) (v16.2.1)
- **Library**: [React](https://react.dev/) (v19)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4)
- **3D & Graphics**: [Three.js](https://threejs.org/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 💻 Getting Started

### Prerequisites
Make sure you have Node.js (v18+) and npm installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rahulkumargit1/Rahul_Kumar-Portfolio.git
   cd Rahul_Kumar-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
